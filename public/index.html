Custom Map Generator

To create a custom map, add the map data to the text box and then click the submit form button. Note that maps are limited to 40 columns and rows.

Select color or monochromatic map:


Map Data Format

Map data should be entered as comma-separated values (CSV).

The fields are:
  1. name
  2. column
  3. row
  4. base economic value
  5. warp destinations

To help get you started, data for the standard map is pre-populated in the form.
